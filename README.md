# Getting Started with this app

## About Project

### Node Version : 16

### React Version : 18.2.0

Untuk menjalankan project menggunakan yarn:

### `1. Install dependency menggunakan perintah: yarn install`

### `2. Jalankan app menggunakan perintah: yarn start`
